  • region: fail to update network gateway (112e7aac8, Qiu Jian)
  • region: cloud provider filter by domain (22486c31b, Qiu Jian)
  • region: check whether guest is nil (024c14d96, rainzm)
  • region: including domain info in project-role results (538097ef7, Qiu Jian)
  • region: scoped policy filter by project id (6afe83e47, Qiu Jian)
