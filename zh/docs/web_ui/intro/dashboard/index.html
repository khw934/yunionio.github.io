控制面板

控制面板用于展示平台的参数指标等信息，便于用户快速了解系统使用情况。

控制面板用于展示平台的参数指标等信息，便于用户快速了解系统使用情况。Cloudpods新版控制面板内置管理后台视图、域管理后台视图和项目视图的默认模板，且支持自定义任意视图下的控制面板的显示指标，针对用户需求定制专属用户的控制面板。

入口：在云管平台单击左上角导航菜单，在弹出的左侧菜单栏中单击 “控制面板” 菜单项，进入控制面板页面。

  • 在控制面板页面，单击指标右上方图标将会跳转到对应的资源页面，如单击虚拟机数量指标右上方图标将会跳转到虚拟机列表页面。

控制面板管理

默认面板暂不支持删除和编辑。控制面板相关指标说明请参考指标。

新建控制面板

当默认控制面板不符合用户需求时，用户可以新建控制面板定制专属用户的控制面板。在不同视图下都可以创建控制面板。

  1. 在控制面板页面，单击控制面板右上角图标，选择下拉菜单 “新建” 菜单项，进入新建控制面板页面。

  2. 设置仪表盘名称并从磁贴库拖拽磁贴进行配置，配置方法如下。

    • 通用配置：任意磁贴都支持、配置、复制粘贴操作、以及放大或缩小磁贴。下面重点介绍如何配置磁贴。

    • 数字图：用于展示资源的数量。

      1. 将数字图的磁贴拖拽到面板的任意位置，单击磁贴的图标，在右侧弹出配置磁贴对话框。

      2. 配置以下参数：

        • 类型：支持设置主机资源和容器资源。
        • 当设置“主机资源” 时，配置以下参数：
          • 磁贴名称：自定义磁贴显示的指标名称。
          • 云环境和平台：选择指标统计的平台，如不选择，则代表全部平台。
          • 区域或云账号：选择指标统计的区域或云账号或调度标签，如不选择，则代表全部区域或全部云账号或全部调度标签。
          • 项目标签：选择指标统计的项目标签进行过滤。
          • 指标：选择具体的指标值，如server代表虚拟机数量等。
          • 当图表类型选择“趋势图”时，配置以下参数：
            • 监控范围： 支持选择近7日或近30日。
          • 当图表类型选择“指标图”时，配置以下参数：
            • 图标：支持选择当前磁贴的图标。
        • 当设置“容器资源”时，配置以下参数：
          • 磁贴名称：自定义磁贴显示的指标名称。
          • 指标：选择容器相关指标，如all.cluster.count表示容器集群数量。

      3. 配置完成后，单击 “确定” 按钮，保存配置。

    • 使用率：用于展示资源的利用率。

      1. 将使用率的磁贴拖拽到面板的任意位置，单击磁贴的图标，在右侧弹出配置磁贴对话框。

      2. 配置以下参数：

        • 类型：支持设置主机资源和容器资源。
        • 当设置“主机资源” 时，配置以下参数：
          • 磁贴名称：自定义磁贴显示的指标名称，如内存利用率等。
          • 云环境和平台：选择指标统计的平台，如不选择，则代表全部平台。
          • 区域或云账号：选择指标统计的区域或云账号，如不选择，则代表全部区域或全部云账号。
          • 总量指标和使用量指标：使用率即使用量与总量的比值。分别选择对应的指标。如总量指标选择host.memory(宿主机内存容量)、使用率选择all.servers.memory(虚拟机内存总量)，两者的比值即内存使用率。
          • 指标名称：根据总量指标和使用量指标自定义磁贴的指标名称，如总量指标选择“宿主机内存容量”，使用量指标选择“虚拟机内存总量”，指标名称可设置为“已使用”。
          • 剩余指标名称：根据总量指标和使用量指标自定义磁贴的剩余指标名称，如总量指标选择“宿主机内存容量”，使用量指标选择“虚拟机内存总量”，剩余指标名称可设置为“未使用”。
          • 当图表类型选择“环形图”时，配置以下参数：
            • 配色方案：支持选择三种环形图的配色方案。
          • 当图表类型选择“水球图”时，无需配置。
        • 当设置“容器资源” 时，配置以下参数：
          • 磁贴名称：自定义磁贴的显示名称，如容器内存使用率。
          • 总量指标和使用量指标：使用率即使用量与总量的比值。分别选择对应的指标。
          • 指标名称：根据总量指标和使用量指标自定义磁贴的指标名称，如总量指标选择“内存总量”，使用量指标选择“内存使用量”，指标名称可设置为“已使用”。
          • 剩余指标名称：根据总量指标和使用量指标自定义磁贴的剩余指标名称，如总量指标选择“内存总量”，使用量指标选择“内存使用量”，剩余指标名称可设置为“未使用”。
          • 当图表类型选择“环形图”时，配置以下参数：
            • 配色方案：支持选择三种环形图的配色方案。
          • 当图表类型选择“水球图”时，无需配置。

      3. 配置完成后，单击 “确定” 按钮，保存配置。

编辑控制面板

该功能用于基于已有的控制面板修改控制面板。可以添加或删除磁贴或更改已配置的磁贴参数等。

  1. 在控制面板页面，单击控制面板右上角图标，选择下拉菜单 “编辑” 菜单项，进入编辑控制面板页面。
  2. 支持修改控制面板名称、或添加或删除磁贴或更改已配置的磁贴参数等。
  3. 修改完成后，单击顶部 “确定” 按钮，完成修改。

导出控制面板

用户可以将已配置好的控制面板导出，并分享给其他用户使用。

  1. 在控制面板页面，单击控制面板右上角图标，选择下拉菜单 “导出” 菜单项，下载控制面板文件（后缀为ocdb）。

导入控制面板

用户可以将导出的控制面板文件导入到平台使用。

  1. 在控制面板页面，单击控制面板右上角图标，选择下拉菜单 “导入” 菜单项，弹出导入对话框。
  2. 将控制面板文件（后缀为ocdb）拖拽到对话框上，单击 “确定” 按钮。
  3. 导入的控制面板将显示在控制面板页面。

克隆控制面板

该功能用于将已有的控制面板复制一份，并显示在控制面板页面中。用户可以基于克隆的控制面板编辑定义自己的面板。

  1. 在控制面板页面，单击控制面板右上角图标，选择下拉菜单 “克隆” 菜单项，复制控制面板。
  2. 克隆的控制面板与原控制面板完全相同，仅名称变成名称-1。

共享控制面板

注意

  • 支持系统管理员操作，支持以admin角色加入项目的用户操作
  • 只有默认面板支持共享，其他不支持
  • 按照视图区分，共享后只影响当前视图

该功能用于将所有人的默认面板更新为当前面板。

  1. 在控制面板页面，单击控制面板右上角图标，选择下拉菜单 “共享” 菜单项，弹出共享对话框。
  2. 单击 “确定” 按钮，完成操作。

重置控制面板

该功能用于重置默认面板，当默认控制面板被修改后，可通过该功能恢复到初始状态。

  1. 在控制面板页面，单击默认控制面板右上角图标，选择下拉菜单 “重置” 菜单项，弹出操作确认对话框。
  2. 单击 “确定” 按钮，完成操作。
